# RTV Tool Kit

  RTV Tool Kit turns Road to Vostok into a live modding workspace.

  This mod adds a full in-game utility UI made for modders, testers, and anyone who wants deeper control while
  experimenting with the game. Instead of guessing what is loaded or digging through files blindly, you can inspect the
  live scene hierarchy, search runtime objects, edit nodes directly, browse save/config data, and poke game systems
  while the game is running.

  Everything is designed to speed up testing, debugging, and discovery.

  ## Features

  - Live scene hierarchy with filter, expand all, and collapse all
  - Object search by name, class, path, group, script, method, or property
  - Selected subtree search for focused debugging
  - Node inspector with metadata, property list, and method calls
  - Property editing directly on the selected node
  - World edit tools for rename, visibility, transform editing, nudging, duplicate, delete, and reparent
  - Helper node creation for quick testing
  - Scene spawning by res:// path
  - Runtime controls for engine state, simulation values, and game data flags
  - user:// file browser with preview and snapshot export
  - Persistent bookmarks, watch list, and selection history
  - Report export, watch report export, and clipboard helpers
  - Toolkit event log for quick feedback while testing

  ## Hotkeys

  - F8 toggles the toolkit overlay
  - F9 refreshes all toolkit views
